03-1 변하는 수, ‘변수’ 이해하기

변수(variable)

  • 다양한 값을 지닌 하나의 속성
  • 데이터 분석의 대상

변수 만들기


변수 만들기

a = 1  # a에 1 할당
a      # a 출력
1

b = 2
b
2

c = 3
c
3

d = 3.5
d
3.5

변수로 연산하기

a + b
3
a + b + c
6
4 / b
2.0
5 * b
10

여러 값으로 구성된 변수 만들기

var1 = [1, 2, 3]
var1
[1, 2, 3]
var2 = [4, 5, 6]
var2
[4, 5, 6]
var1 + var2
[1, 2, 3, 4, 5, 6]

문자로 된 변수 만들기

str1 = 'a'
str1
'a'
str2 = 'text'
str2
'text'
str3 = 'Hello World!'
str3
'Hello World!'

여러 문자로 된 변수 만들기

str4 = ['a', 'b', 'c']
str4
['a', 'b', 'c']
str5 = ['Hello!', 'World', 'is', 'good!']
str5
['Hello!', 'World', 'is', 'good!']

문자 변수 결합하기

str2 + str3
'textHello World!'
str2 + ' ' + str3
'text Hello World!'

문자로 된 변수로는 연산할 수 없다

str1 + 2
Error in py_call_impl(callable, dots$args, dots$keywords): TypeError: can only concatenate str (not "int") to str

03-2 마술 상자 같은 ‘함수’ 이해하기

함수(function)

  • 값을 넣으면 특정한 기능을 수행해 처음과 다른 값으로 만듦

함수 이용하기

# 변수 만들기
x = [1, 2, 3]
x
[1, 2, 3]
# 함수 적용하기
sum(x)
6
max(x)
3
min(x)
1

함수의 결과물로 새 변수 만들기

x_sum = sum(x)
x_sum
6
x_max = max(x)
x_max
3

03-3 함수 꾸러미, ‘패키지’ 이해하기

패키지(packages)

  • 함수가 여러 개 들어 있는 꾸러미
  • 함수 사용하려면 패키지 설치 먼저
  • 아나콘다에 주요 패키지 대부분 들어있음

패키지 로드하기
import seaborn
패키지 함수 사용하기
var = ['a', 'a', 'b', 'c']
var
['a', 'a', 'b', 'c']
seaborn.countplot(x = var)

패키지 약어 활용하기

import seaborn as sns
sns.countplot(x = var)
